A Papirus folder colour for the [neo-candy-icons](https://github.com/erikdubois) icon theme. Ships only the **neo-candy-papirus-indigo** folder (places) icons in the **indigo** colour and inherits the base set, so `neo-candy-icons-git` is required. Folders sourced from [PapirusDevelopmentTeam/papirus-icon-theme](https://github.com/PapirusDevelopmentTeam/papirus-icon-theme).

## Installation (Arch / Kiro — nemesis_repo)

```
sudo pacman -S neo-candy-papirus-indigo-icons-git
```
